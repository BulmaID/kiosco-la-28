# Kiosco La 28 — Primera versión

## Cómo abrir la web
1. Descomprimí el archivo ZIP.
2. Entrá en la carpeta `kiosco_la_28_web`.
3. Tocá dos veces `index.html`.
4. Se abrirá en tu navegador.

## Archivos principales
- `index.html`: contenido y estructura.
- `styles.css`: colores, diseño y adaptación a celular.
- `script.js`: menú móvil y año automático.
- `assets/`: fotos y logo.

## Primer ejercicio
Abrí `index.html` en Visual Studio Code y buscá:
`Todo lo que necesitás, cerca de casa.`

Cambiá temporalmente esa frase, guardá el archivo y actualizá el navegador.
Después volvé a dejar la frase original.

Ese cambio pequeño te permite comprobar cómo HTML controla el contenido visible.
